# se109a-blog

a simple blog web application - Node.js/JavaScript(HTML+CSS)/MongoDB

test: mocha, chai, supertest, puppeteer


## Setup

```
npm install
```

## Run

run backend server

```
npm start
```

run frontend server

```
npm run client
```

## Test

API test

```
npm run test:api
```

E2E test

```
npm run test:e2e
```
